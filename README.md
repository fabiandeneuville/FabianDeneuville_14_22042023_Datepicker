# FD React Datepicker

## Installation :

```
npm i fd-react-datepicker
```

## Use :

```
import DatePicker from 'fd-react-datepicker';
```

```
    <DatePicker
        label={'Date'}
        required={true}
        onChange={(date) => {}}
    />
```
label, required and onChange props are required.
Define your own onChange function to handle the date value.
