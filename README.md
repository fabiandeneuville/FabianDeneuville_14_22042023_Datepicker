# FD React Datepicker


## Basic React + Typescript DatePicker

### Installation :

```
npm i fd-react-datepicker
```

### Use :

```
import DatePicker from 'fd-react-datepicker';
```

```
    <DatePicker
        label={'Date'}
        required={true}
        onChange={(date) => {}}
    />
```

### Props :

-   label: string - required
-   required: boolean - required
-   onChange: function - required
