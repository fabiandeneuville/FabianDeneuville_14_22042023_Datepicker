# FD React Datepicker


## Basic React + Typescript DatePicker

### Installation :

```
npm i fd-react-datepicker
```

### Use :

```
import DatePicker from 'fd-react-datepicker';
```

```
    <DatePicker
        required={true}
        onChange={(date) => {}}
    />
```

### Props :

#### Required props : 
-   required: boolean 
-   onChange: function

#### Optional props: 
-   label: string
-   style: object
-   className: string
-   id: string
-   name: string
-   value: string (yyyy-mm-dd)
